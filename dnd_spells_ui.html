🪄 D&D Spell Bindings

Key / ButtonModifiersWheelSlotDelay (ms)

Click the key/button cell to capture a new binding — press any key or mouse button (Esc to cancel).
Slots are 1=top, 2=upper-right, 3=lower-right, 4=lower-left, 5=upper-left (clockwise from 12 o'clock).
Mouse 3=middle click, Mouse 4=first thumb button, higher numbers=extra side buttons.
Multi-cast: add two (or more) rows with the same key to fire multiple spells back-to-back from one press — e.g. Q1 + E1 on the same button loads slot 1 in both hands for a merge cast.
Delay (ms): wait this long before firing this row. Row 1's delay = hold-then-fire. Rows 2+'s delay = gap between casts in a sequence. 0 = fire immediately.
Hold to commit, release to cancel: releasing the button mid-cast aborts the macro and returns control of the mouse instantly. Hold long enough for the whole sequence (including any delays) to complete.
